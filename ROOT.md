# Workspace Root

This workspace is the starting point for **EdgeQuanta**, a product concept built around infrastructure and access layers for real superconducting quantum hardware.

## Current Files

- `README.md` — project overview and product framing
- `bitquanta.md` — translated protocol document describing the underlying backend capabilities

## Current Understanding

The protocol suggests the backend already supports:

- circuit/job submission
- async execution tracking
- result return and result recovery
- live event subscriptions
- calibration and benchmarking visibility
- chip configuration queries
- exclusive machine-time reservation

## Working Product Direction

EdgeQuanta should be approached first as a:

- quantum infrastructure platform
- quantum access gateway
- observability and scheduling layer

rather than as a pure algorithm or consumer application.
