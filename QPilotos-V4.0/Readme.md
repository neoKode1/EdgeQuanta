# 本源司南 (Origin PilotOS) 安装包说明

欢迎使用本源司南量子操作系统。本压缩包包含了系统自动部署程序、测控模拟服务示例以及相关的技术文档。

本文档旨在帮助您快速了解目录结构，并指引您完成系统的安装部署与开发对接。

## 📂 目录结构说明

本安装包解压后包含以下文件和目录：

| 文件/目录名                 | 说明                                                         |
| :-------------------------- | :----------------------------------------------------------- |
| **pilotos-4.0.tar.gz**      | **自动部署软件包**。包含司南核心系统及自动化部署脚本 (`AutoDeploy.py`)。 |
| **python_simulator.tar.gz** | **异构量子芯片测控模拟服务**。提供测控参考示例，便于用户对接自有硬件。 |
| **本源司南用户手册.pdf**    | 系统完整用户手册，包含详细的安装步骤（第2章）及功能说明。    |
| **模拟服务使用介绍.pdf**    | 指导如何运行和使用 `python_simulator` 模拟服务。             |
| **异构量子芯片对接协议/**   | （文件夹）包含测控服务与司南系统对接的具体通讯协议文档。     |

## 🚀 快速开始

### 1. 系统安装部署 (System Deployment)

本源司南提供了一键自动化部署脚本，请按照以下步骤操作：

1.  **解压软件包**：
    将 `pilotos-4.0.tar.gz` 解压到目标服务器。
    
    ```bash
    tar -zxvf pilotos-4.0.tar.gz
    cd pilotos-4.0
    ```
    
2.  **执行自动部署**：
    在解压后的目录中，运行自动化部署脚本。
    
3.  **详细指引**：
    * 部署过程中的环境依赖、参数配置及常见问题，请务必参考 **《本源司南用户手册.pdf》中的“第二章”**。

### 2. 测控服务对接与模拟 (Measurement & Control Integration)

如果您需要将自有的量子芯片测控系统接入本源司南，或进行开发调试，请使用我们提供的模拟服务。

* **运行模拟服务**：
    解压 `python_simulator.tar.gz`，并参考 **《模拟服务使用介绍.pdf》** 启动服务。该服务展示了标准的测控响应逻辑。

* **对接协议开发**：
    如需对接真实的物理测控设备，请查阅 **`异构量子芯片对接协议/`** 目录下的文档。该文档详细定义了司南系统与底层测控硬件的通信接口规范。
